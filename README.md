# Student Score Predictor

A Machine Learning web app that predicts student scores based on input features like study hours, age, and other factors. Built using Streamlit and deployed for real-time predictions.

---

Dataset collected from kaggle:
The link below:-
https://www.kaggle.com/datasets/sarveshchhetri/student-lifestyle-vs-academic-performance-dataset

## Live Demo

👉 Try the app here:  
🔗 https://studentscorepredictor-itwo73j4uhransjpcs7nnb.streamlit.app/

---

## Features

- Predict student scores based on stress level,screen time,sleep hours, previous GPA etc.
- Machine Learning model (XGBoost)
- Interactive UI using Streamlit
- Fast and lightweight deployment
- Visual insights (feature importance, graphs)

---

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
